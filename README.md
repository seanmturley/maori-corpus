# Māori Corpus

A cleaned version of the 2017 Māori 100K sentence corpus from the Leipzig Corpora Collection.<sup id="r1">[1](#f1)</sup>

The original corpus data is available on the [Wortschatz Leipzig website](https://wortschatz.uni-leipzig.de/en/download/mri), as well as similar corpora for many [other languages](https://wortschatz.uni-leipzig.de/en/download/).

Any dirivitive works should also cite [the original paper](http://www.lrec-conf.org/proceedings/lrec2012/pdf/327_Paper.pdf), as below.

<b id="f1">1.</b> D. Goldhahn, T. Eckart & U. Quasthoff: Building Large Monolingual Dictionaries at the Leipzig Corpora Collection: From 100 to 200 Languages.
In: _Proceedings of the 8th International Language Resources and Evaluation (LREC 2012), 2012._ [↩](#r1)
